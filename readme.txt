---------SCRIPTS-------
-----------------------
collect_data.py
---------------
- Collect data from the internet (e.g: from json files)
- Create csv and json files:
	- countries
	- cities
	- capital_cities
	- landmarks
	- actors
	- movies
	
--------------------------------------------------------
generate_queries.py --mode <mode> --category <cat> --difficulty [1-7] --amount <amount>
--------------------------------------------------------
- Generate queries from csv files given game mode, category and difficulty
[game mode: bubble matcher/bubble clicker]

-------------
load_data.csv
-------------
- Load data to SQL db given csv files
